Setup
Quick Start Guide
Get running in under 5 minutes.
1
Drop In
Copy the resources/[frostline] folder into your FiveM server-data resources directory.
2
Configure
Edit server.cfg — set your license key, admin identifiers, and Discord webhook URLs.
3
Customize
Edit each module's config.lua to set locations, vehicles, branding text, and whitelist entries.
4
Launch
Run start.bat and connect. All modules auto-load in the correct order.
